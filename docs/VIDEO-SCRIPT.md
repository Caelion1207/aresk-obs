# Script de Video Tutorial: ARESK-OBS

## Introducción (0:00-0:15)
Bienvenido a ARESK-OBS, un instrumento de medición de estabilidad cognitiva en sistemas de inteligencia artificial. Este tutorial te mostrará cómo usar las funcionalidades principales del sistema.

## Página Principal (0:15-0:45)
Al abrir ARESK-OBS, encontrarás la página principal con acceso a todas las herramientas. Desde aquí puedes navegar a tres secciones principales: el Experimento de Estabilidad, el Diagrama de Arquitectura, y el Core Dashboard.

## Experimento de Estabilidad (0:45-1:30)
Haz clic en "Experimento de Estabilidad" para ver el análisis temporal del Régimen A. Aquí encontrarás una gráfica interactiva que muestra la evolución de tres métricas clave: Omega (coste de control), Epsilon (entropía semántica), y V (Lyapunov). Puedes cambiar entre métricas usando los botones superiores. La línea roja punteada marca el umbral crítico de 0.5. El turno 13 está destacado porque alcanzó el máximo valor de Omega.

## Tabla de Datos (1:30-2:00)
Debajo de la gráfica, expande la tabla de datos para ver los 50 turnos completos con sus métricas. Puedes ordenar por cualquier columna haciendo clic en los encabezados. El turno 13 aparece destacado en color ámbar.

## Comparación de Regímenes (2:00-2:45)
Haz clic en "Comparar con otros regímenes" para ver la gráfica comparativa. Esta visualización superpone las métricas Omega de tres regímenes diferentes: Régimen A (alta entropía, en azul), Régimen B (ruido medio, en morado), y Régimen C (CAELION activo, en verde). Las tarjetas inferiores muestran estadísticas detalladas de cada régimen.

## Diagrama de Arquitectura (2:45-3:30)
Regresa a la página principal y haz clic en "Diagrama de Arquitectura". Este diagrama interactivo muestra el flujo de datos entre los 8 componentes del sistema. Pasa el cursor sobre cualquier componente para ver su descripción. Haz clic en un componente para abrir la documentación técnica detallada con especificaciones, métricas de rendimiento y ejemplos de código.

## Core Dashboard (3:30-4:00)
El Core Dashboard muestra métricas en tiempo real del sistema, incluyendo el estado de la cadena de auditoría. Aquí puedes ver el hash del génesis, el número total de logs, y la validez de la cadena. El sistema se actualiza automáticamente cada 5 segundos.

## Conclusión (4:00-4:15)
ARESK-OBS es un instrumento no predictivo y no autónomo diseñado para medir estabilidad cognitiva. Explora las diferentes secciones para analizar métricas, comparar regímenes y entender la arquitectura del sistema. Gracias por usar ARESK-OBS.
